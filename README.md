## Running

#### Clone & install

* Clone this repo `git clone git@github.com:thaiinhk/VocabReactNative.git`
* `cd VocabReactNative`
* run `npm install`

#### iOS

* Open `Vocab.xcodeproj` in `XCode`
* Press `cmd+r` to build it

#### Android

* Run `android avd` and start an emulator
* Run `react-native run-android`
